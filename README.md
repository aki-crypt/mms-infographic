# MMS (Mental Management School) 講師プロジェクト

このプロジェクトは、MMS（メンタルマネジメントスクール）で講師を務めるにあたり、必要な情報や考えを整理・蓄積するための場所です。

## 基本方針
1. **すべてMarkdown形式で保存**: 情報や資料は原則として `.md` ファイルで管理します。
2. **追記型の履歴管理**: 考え方の履歴やプロセスを大切にし、基本的に情報は削除せず、追記・蓄積していくアプローチを取ります。
3. **プロンプトの記録**: 今後AIとのやり取りで使用したプロンプトや、最初の要望なども記録として残し、思考のプロセスを後から振り返れるようにします。

## 主要ファイル
- `README.md`: プロジェクトの概要と基本方針（このファイル）
- `thoughts_history.md`: 考え方の履歴、アイデアの推移などを時系列で追記していくファイル。
- `prompts_log.md`: 使用したプロンプトや要望の履歴を保存するファイル。
